# Bogotá Beans

Landing page para una cafetería de especialidad en Bogotá.

## Características

- Diseño moderno y responsivo.
- Paleta de colores cálidos y relacionados con el café.
- Secciones principales: Inicio, Productos, Nosotros, Impacto y Contacto.

## Tecnologías

- HTML5
- CSS3
- JavaScript

## Instalación y Uso

1. Clona el repositorio:
   ```bash
   git clone https://github.com/alejandra-bolanos-ldg/landing-page.git
   ```
2. Abre el archivo `index.html` en tu navegador.
